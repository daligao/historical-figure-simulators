## How it works

Each simulator is a standalone HTML file powered by an LLM API. It:
- Adopts the historical figure's documented vocabulary, worldview, and writing style
- Stays grounded in historically accurate positions
- Adds conversational depth by referencing the figure's actual works and known opinions
- Runs entirely in the browser — no backend, no data collection

---

## Run locally

```bash
git clone https://github.com/daligao/historical-figure-simulators
cd historical-figure-simulators
open buffett-simulator.html   # or any other
```

Each file is self-contained. Just open in a browser. You'll need an API key for the LLM backend (set in the UI).
